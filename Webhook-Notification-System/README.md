# Webhook Notification System

This project demonstrates a webhook sender and listener system built using Python and Flask.

## Features
- Webhook sender
- Webhook listener
- Event handling
- Notification processing

## Technologies Used
- Python
- Flask
- Requests

## Project Structure
- Listener/
- Sender/

## How to Run
Install dependencies:
pip install -r requirements.txt

Run the listener:
python app.py

Run sender:
python send_webhook.py
